# 炫锋商行官网预览

这是炫锋商行官网预览版本，定位为：

- 面向散人玩家的官网首页
- 传奇手游搬砖攻略中心
- 装备材料价值判断与出售路径说明
- 新手路线和客服承接入口

## 页面结构

```text
/                         官网首页
/guide/                   攻略中心
/guide/articles/          攻略文章
/robots.txt               抓取规则
/sitemap.xml              站点地图
```

## 主要内容

- 散人玩家适合人群说明
- 加入前的三步流程
- 商行能提供的攻略、社群、客服和出售对接流程
- 合作前的风险和边界说明
- 传奇手游搬砖入门文章
- 新手起号流程文章

## 说明

页面内容仅用于业务介绍和资料参考，不承诺固定结果。玩家应根据自身时间、设备和基础情况理性判断。
